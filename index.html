Описание проекта

Учебный проект "Code Translator" разработан в 2026 году для перевода кода между четырьмя языками программирования: Python, Pascal, Java и C++. Проект представляет собой полностью статическое веб-приложение, работающее без сервера.

Работает в любом браузере
Не требует установки
Поддерживает 12 направлений перевода

Проект выполнен под руководством Шевцова Даниила
